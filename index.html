English Buddy

日常会話・ビジネス・IT用語まで、これからの英語学習をまとめてサポート。

Daily

Nice to meet you.

はじめまして。

シーン: 自己紹介

1. 日常会話の土台

  • 朝・昼・夜のあいさつ
  • 相づちとリアクション
  • 短い自己紹介

まずは「言える安心感」を作るのが最優先。

2. ビジネス英語の型

  • ミーティングでの発言
  • メールの定番フレーズ
  • 依頼・報告・相談

型を覚えると応用がしやすい。

3. IT業界の頻出用語

  • Requirement / Spec / Deadline
  • Deploy / Release / Rollback
  • Stand-up / Sprint / Backlog

現場でよく耳にする単語からスタート。

今日の学習プラン（30分）

  1. 5分: フラッシュカードで声に出す
  2. 10分: Dailyフレーズを3つ使って会話を作る
  3. 10分: Businessフレーズをメール風に書く
  4. 5分: IT用語を1つ調べて例文を書く

コツ

「完璧に話す」より「伝わる」を意識する。

毎日続けられる量に分割する。

次に作りたい機能は？

ボタンを押すと次の学習テーマが表示されます。
